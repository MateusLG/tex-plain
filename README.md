# tex-plain

Gerador local e open-source de documentos LaTeX a partir de resumos escritos e templates (artigo, anotação, estudo dirigido, etc.), com IA fechando o loop de compilação.

## Ideia

Entrada: resumo em texto + escolha de template.
Saída: `.tex` gerado, compilado para PDF, com auto-correção em caso de erro de compilação (pacotes faltando, encoding, citações).
